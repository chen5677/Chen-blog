## 为什么要写博客？

写博客是一个很好的学习和沉淀方式。通过将知识整理成文字，可以加深理解，
也能帮助到其他有相同疑问的人。

## 技术选型

这个博客使用 **Pelican** 构建，这是一个基于 Python 的静态站点生成器。

### 为什么选择 Pelican？

1. **Python 生态** — 使用熟悉的 Python 工具链
2. **Markdown 写作** — 专注于内容，而非排版
3. **静态站点** — 部署简单，加载速度快
4. **GitHub Pages 免费托管** — 零成本运维

### 代码示例

```python
def greet(name: str) -> str:
    """一个简单的问候函数"""
    return f"Hello, {name}!"

print(greet("World"))
# 输出: Hello, World!
```
